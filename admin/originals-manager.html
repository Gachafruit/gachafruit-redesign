Gachafruit Studio

Originals Manager

Manage the Current Project, Etsy section settings, and Free Models tiles. Export JSON when ready and place it at /data/originals-content.json.

Current Project

The featured editorial entry shown at the top of /originals/. One project is shown at a time. Archive it when the next project begins.

Etsy Section

Controls the Shop Our Originals preview row on /originals/. Listings are pulled from the same API used by the Explore All page — no separate setup needed.

Free Models

Manual tiles for the Browse the Community Archive section. Each tile links to an external download (Printables, Thingiverse, etc.). No price displayed — "Download" button by default.
